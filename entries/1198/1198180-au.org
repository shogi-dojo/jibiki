- UK :: Дієслово 会う (зустрічатися) є неперехідним і керує відмінком з часткою に для позначення особи, з якою зустрічаються (наприклад, 友達に会う — зустрітися з другом). Для спільної взаємної дії також може вживатися частка と (友達と会う — зустрічатися з другом). Для розрізнення відтінків значення на письмі вживають різні кандзі: 会у — нейтральний варіант для зустрічі з людьми; 遭у — для неприємних чи небажаних подій (наприклад, 事故に遭у — потрапити в аварію); 逢у — для доленосних чи емоційно важливих зустрічей; 遇у — для випадкових зустрічей.
- LEVEL :: beginner
- REGISTER :: neutral
