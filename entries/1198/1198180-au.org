*** note-s-1198180-001-001
:PROPERTIES:
:END:
- UK :: Дієслово 会う (зустрічатися) є неперехідним і керує відмінком з часткою に для позначення особи, з якою зустрічаються (наприклад, 友達に会う — зустрітися з другом). Для спільної взаємної дії також може вживатися частка と (友達と会う — зустрічатися з другом). Для розрізнення відтінків значення на письмі вживають різні кандзі: 会う — нейтральний варіант для зустрічі з людьми; 遭う — для неприємних чи небажаних подій (наприклад, 事故に遭う — потрапити в аварію); 逢う — для доленосних чи емоційно важливих зустрічей; 遇う — для випадкових зустрічей.
